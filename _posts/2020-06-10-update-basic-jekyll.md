## to Change Your Jekyll Theme
1. Open your Gemfile
2. Add the theme to the Gemfile 
3. `gem "jekyll-theme-example"`
4. Open your terminal
5. Install theme using the following command
    - `bundle update` -> sudo
    - `bundler install` -> sudo ?
    - `bundle exec jekyll serve`
